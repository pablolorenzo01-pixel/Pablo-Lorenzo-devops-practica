# Practica 02
## Pablo Lorenzo Souto
### El código actúa como una simulación completa del sistema de una tienda online, demostrando la funcionalidad de sus componentes principales al iniciar el servicio de la tienda, registrar varios tipos de usuarios (clientes y un administrador), y llena el inventario con productos diversos, incluyendo artículos electrónicos con garantía y ropa con talla y color específicos. El script procede a simular el proceso de compra mediante la creación de múltiples pedidos por parte de los clientes, gestionando la transacción y el descuento de la cantidad pedida del inventario; finalmente, verifica la integridad del sistema al listar el stock actualizado después de los pedidos y al consultar el historial de compras de un cliente en particular, validando así todo el flujo de negocio.
